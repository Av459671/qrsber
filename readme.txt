Пара программ предназначенных для генерации платежного QR согласно ГОСТ.

Для обеих  программ общий файл sberqr.ini
Из этого файла берутся банковские реквизиты получателя платежа, 
пути к папкам с входяищими файлами в формате csv и исходящими файлами с QR кодами для оплаты в формате png.
Из примера ini файла все понятно:

[Acc]
Name=ООО "Электронный проездной"
PersonalAcc=40702810127000005314
BankName=КИРОВСКОЕ ОТДЕЛЕНИЕ N8612 ПАО СБЕРБАНК
BIC=043304609
CorrespAcc=30101810500000000609
KPP=434501001
PayeeINN=4345312512
[Path]
incsv=p:\QR_Sber\in\
outqr=p:\QR_Sber\out\

Важно. В путях ставить всегда последний слэш.

Программа genSberQR.exe
Работает в пакетном режиме. 
Сканирует по пути incsv файлы по маске transactions*.csv
Файлы csv должны быть в формате указанном в примере transactions_primer.csv
Используются колонки:
Reference number - номер транспортной карты;
Номер карты - номер банковской карты;
terminal_date - дата и время.
Соответсвие формата внутри не контролируется!
Реквизиты получателя платежа берутся из sberqr.ini
Сумма плтатежа указывается равной нолю - плательщик ее указывает сам.
Выходные файлы с QR кодом помещаются в папку по пути outqr.
Имя файла соответсвует номеру транспортной карты.
Одинаковые файлы автоматически перезаписываются.
Действия пользователя.
1. В папку по пути указанном в incsv (это в sberqr.ini) помещаем файлы csv для обработки.
2. Запускаем программу genSberQR.exe (на экране будет отображаться консоль).
3. Выходные файлы с QR кодом будут в папке по пути указанном outqr (это в sberqr.ini)
4. Фалы из входной и выходной папки перемещаем (или удаляем) куда нгам нужно.

Программа SberQR.exe
Работает в диалоговом режиме и позволяет сформировать единичный файл с QR кодом.
Реквизиты получателя платежа и путь к выходному файлу берутся из sberqr.ini
Действия пользователя.
1. Запускаем программу SberQR.exe. На экране будет отображаться форма для ввода данных.
2. Вводим исходные данные. Важно! Сумму вводить в копейках без разделительных знаков. Это тербование ГОСТ.
3. Нажимаем кнопку "Создать QR"
4. Файл с QR кодом будет в папке по пути указанном outqr (это в sberqr.ini). Путь будет указан в сообщении.
